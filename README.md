# Mundina Nildana
Mundina Nildana is a full-stack web application developed using Node.js and Express.js, leveraging EJS for server-side rendering and following the MVC (Model–View–Controller) architecture. This system enables users to authenticate and authorize through secure sign-up and login functionality, manage property listings by adding and editing properties, submit reviews, and view property locations on an interactive map.

### Live Website: https://mundinanildana.onrender.com
